## 📋 Как запустить тесты (ручное выполнение)

### Требования к окружению

- ✅ **VPN** — обязательно включён
- ✅ Браузер Google Chrome (рекомендуется)
- ✅ Стабильное интернет-соединение

### Инструкция

1. **Включить VPN**
2. Открыть сайт: `https://qa-internship.avito.com`
3. Открыть файл [`TESTCASES.md`](./TESTCASES.md)
4. Выполнить шаги каждого тест-кейса
5. Сравнить фактический результат с ожидаемым
6. При обнаружении расхождений — сверить с [`BUGS.md`](./BUGS.md)

### Для тестирования мобильной версии

- Нажать `F12` → значок телефона (Device Toolbar)
- Выбрать `iPhone 12` или `Pixel 5`
