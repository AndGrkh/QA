# Тест-кейсы для UI тестирования (стажировка QA, весенняя волна 2026)

## Описание
Тестирование сайта https://qa-internship.avito.com

### Что тестируется:
- **Десктоп**: фильтр «Диапазон цен», сортировка «По цене», фильтр «Категория», тогл «Только срочные»
- **Десктоп**: страница статистики — таймер обновления (кнопки Обновить, Старт, Стоп)
- **Мобильная версия**: переключение тёмной/светлой темы

### Техники тест-дизайна:
- Классы эквивалентности
- Анализ граничных значений
- Предугадывание ошибок

## TC-01: Фильтр «Диапазон цен» — позитивный сценарий

| Шаг | Действие | Ожидаемый результат |
|-----|----------|----------------------|
| 1 | В поле «Цена от» ввести 1000 | Поле принимает значение 1000 |
| 2 | В поле «Цена до» ввести 50000 | Поле принимает значение 50000 |
| 3 | Нажать «Применить» | Показываются объявления с ценой от 1000 до 50000 |

**Результат**: ✅ ПРОЙДЕН

## TC-02: Фильтр «Диапазон цен» — граничные значения

| Шаг | Действие | Ожидаемый результат |
|-----|----------|----------------------|
| 1 | «Цена от» = 0, «Цена до» = 0 | Объявления с ценой 0 или пустой список |
| 2 | «Цена от» = 999999999, «Цена до» = 999999999 | Пустой список |
| 3 | «Цена от» = 500, «Цена до» = 100 | Ошибка «Цена «до» не может быть меньше цены «от»» |
| 4 | «Цена от» = abc, «Цена до» = xyz | Поля не принимают буквы |

**Результат**: ✅ ПРОЙДЕН
