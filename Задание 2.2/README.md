## Как запустить тесты (ручное тестирование)

### Требования:
- ✅ Включён VPN
- ✅ Браузер Chrome (рекомендуется)
- ✅ Стабильный интернет

### Инструкция:

1. **Включить VPN**
2. Открыть сайт: `https://qa-internship.avito.com`
3. Открыть файл `TESTCASES.md`
4. Выполнить шаги каждого тест-кейса
5. Сравнить фактический результат с ожидаемым
6. Если результат не совпадает → это баг (все баги описаны в `BUGS.md`)

### Для тестирования мобильной версии:
- Нажать `F12` → значок телефона (Device Toolbar)
- Выбрать `iPhone 12` или `Pixel 5`
